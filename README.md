# BorçDefteri

Basit ve modern bir borç/alacak takip uygulaması.

## Özellikler

- Kişi ekleme
- Borç / alacak takibi
- Hareket geçmişi
- Gece modu
- Özel silme onayı
- LocalStorage ile veri saklama
- Responsive tasarım

## Teknolojiler

- React
- CSS3
- LocalStorage
